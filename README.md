# 🎬 Netflix Data Visualization

## 🚀 Project Overview

This project focuses on visualizing Netflix Movies and TV Shows data using Python, Pandas, Matplotlib, and Seaborn.

## 📊 Features

- Movies vs TV Shows Analysis
- Top 10 Countries Analysis
- Content Added Over Years
- Ratings Distribution
- Release Year Distribution

## 🛠️ Technologies Used

- Python
- Pandas
- Matplotlib
- Seaborn

## 📂 Dataset

Netflix Movies and TV Shows Dataset containing:

- Title
- Type
- Country
- Rating
- Release Year
- Date Added

## 📈 Output

The visualization includes:

1. Movies vs TV Shows
2. Top 10 Countries
3. Content Added Over Years
4. Ratings Distribution
5. Release Year Distribution

## 🎯 Conclusion

Data visualization helps understand Netflix content trends and distribution patterns effectively.
